Här skapar du en mätning på några minuter och ser utveckling över tid.

INFO
Formulär
Frågor + svarsalternativ (källan för alla mätningar).
Mätningar
Datum + antal per svar (sparas lokalt).
Rapport
Grafer + jämförelse mot föregående.

Status

Visar om formulär och mätningar finns sparade i webbläsaren.

OUTPUT
Läser status…
Tips: Om status säger “inget aktivt formulär” → gå till Formulär och aktivera.

Arbetsflöde

Gör alltid i denna ordning för att få jämförbar statistik.

PROCESS
1) Formulär
Säkerställ att rätt frågor/svar används.
2) Ny mätning
Skriv datum + antal per svarsalternativ.
3) Rapport
Se grafer + “utveckling från förra mätningen”.

Snabblänkar

Gå direkt till rätt vy.

NAV
Formulär
Se/aktivera frågor och svarsalternativ.
Ny mätning
Registrera datum + antal per svar.
Rapport
Grafer + jämförelse mot föregående.
